## What is Muximux?

Muximux is a dashboard that aggregates multiple web applications into a single tabbed interface. Perfect for homelab users running self-hosted services like Plex, Sonarr, Radarr, and more.

## Goals

- Modern tech stack with improved developer experience
- Maintain the lightweight, zero-database philosophy
- Enhanced UI/UX with responsive design
- Easy deployment (single binary or container)
